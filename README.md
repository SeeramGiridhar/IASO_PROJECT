# IASO Med

IASO Med is a Vite React healthcare platform with patient and doctor flows, Supabase database support, and Vercel deployment config.

The project includes:

- Patient and doctor authentication
- Medical report upload and AI analysis flow
- Report archive and analysis screens
- Doctor search and appointment booking
- Doctor dashboard, schedule, profile, and report review
- Supabase schema with RLS policies
- Vercel SPA routing config

## Local Development

```powershell
npm.cmd install
npm.cmd run dev
```

## Build

```powershell
npm.cmd run build
```
